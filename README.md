# Парсер КТРУ с сайта zakupki.gov.ru


Parser KTRU это скрипт для проверки характеристик на соответствие определенному КТРУ.

## Установка

Скачать последнюю версию можно тут https://github.com/Babloved/parser_ktru_zakupki_gov_ru/releases.


## Использование

При запуске, надо будет подготовить буфер обмена, заполнив его таблицей из Word или друго текстового редактора<br />
![image](https://github.com/user-attachments/assets/d70437f6-8b3f-45ee-9647-0ebe694d01fd)

Таблица должна содержать три столбца: наименование характеристики из КТРУ, требуемое значение и единицу измерения (единица измерения может быть пустой)
```cmd
                      ПАРСЕР КТРУ V0.x
Чтобы считать характеристики из буфера обмена нажмите Space
```
Нажимаем на пробел

```cmd
Обрабатываю данные с буфера
100%|███████████████████████████████████████████████████████████████████████████████| 36/36 [00:00<00:00, 36157.79it/s]
Введите номер КТРУ
```
Если на этом нет предупреждений то буфер полностью отвечает структуре и можно смело ввести требуемый КТРУ формата x.x.x.x-xxxxxxxx (Пример 26.30.11.110-00000001) и нажать Enter<br />
Если есть ошибки скорее всего в ячейках ХАРАКТЕРИСТИКА ИЗ КТРУ или ТРЕБУЕМОЕ ЗНАЧЕНИЕ пустые поля и такого не может быть, перезапуск на данном этапе пока не предусмотрен требуется ввести КТРУ и продолжить
```cmd
Обрабатываю данные с буфера
  0%|                                                                                            | 0/6 [00:00<?, ?it/s]
ХАРАКТЕРИСТИКА:"тановка" ИЛИ ЗНАЧЕНИЕ:"
" ПУСТОЕ, ЭТО ОШИБКА!
Возможно непредсказуемое поведение, лучше исправить опечатку, строка: 1
ХАРАКТЕРИСТИКА:"вдали" ИЛИ ЗНАЧЕНИЕ:"
" ПУСТОЕ, ЭТО ОШИБКА!
Возможно непредсказуемое поведение, лучше исправить опечатку, строка: 2
100%|██████████████████████████████████████████████████████████████████████████████████| 6/6 
Введите номер КТРУ
```
